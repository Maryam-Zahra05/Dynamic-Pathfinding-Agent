# Dynamic Pathfinding Agent

This project implements a dynamic pathfinding agent using informed search algorithms
(A* and Greedy Best-First Search) in a grid-based environment with real-time obstacle spawning.

## Features
- Adjustable grid size
- Interactive obstacle editing
- Random map generation
- A* and GBFS algorithms
- Manhattan and Euclidean heuristics
- Dynamic obstacles with re-planning
- Real-time visualization and metrics

## Requirements
- Python 3.x
- Tkinter (comes pre-installed with Python)
